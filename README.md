# Google-Machine-learning-crash-course
这个资源涵盖了谷歌机器学习速成课程（中文版）的所有内容，主要是为了方便国内机器学习爱好者学习这门课程。内容主要以加利福尼亚房价预测为线索，讲解了使用机器学习和深度学习解决现实世界的一般方法。
